Vinilo

invisible

Vinilo es una aplicación web que permite conectar a músicos independientes con organizadores mediante la creación de eventos para que se puedan reunir músicos, organizadores y público interesado en música nueva. Se implemento el codigo de la aplicación web utilizando el ambiente de desarrollo IntelliJ IDEA y las tecnologías de front-end HTML, CSS y JavaScript y las tecnologías de back-end Java y Play Framework.

MedicAndes

invisible

MedicAndes es una aplicación web que busca mejorar el proceso que tiene que hacer los pacientes en postoperatorio o rehabilitación, dado a que deben ir al hospital una vez por semana para realizarse un conjunto de exámenes de acuerdo a su cardiopatía y así saber el estado de salud actual y ajustar el tratamiento de ser necesario. El hospital ve este proceso como algo dispendioso y complicado ya que cerca del 50 % de sus pacientes tiene que cambiar el horario de sus exámenes durante la semana y casi un 80 % de ellos cancela una cita por lo menos una vez al mes. La inasistencia afecta el plan de rehabilitación de los pacientes y los pone en riesgo de recaídas de salud o nuevas condiciones producto de la desatención. La aplicación web adicionalmente esta conectada a unos brazaletes inteligentes que llevan los pacientes que recopilan información cardiaca relevante para las diferentes cardiopatías (i.e., frecuencia cardiaca, presión sanguínea, nivel de estrés).De aqui la información es llevaba a la aplicacion para que el paciente pueda ver el historial de sus mediciones y generar alerta en el hospital en caso de emergencia. Tambien la aplicación permite comunicacion entre el medico encargado y el paciente correspondiente para que este pueda darle consejos de salud en caso de ser necesario.
